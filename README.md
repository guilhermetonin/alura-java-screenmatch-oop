# Carreira | Java: aplicando a Orientação a Objetos

> O curso explora os pilares da Programação Orientada a Objetos usando a linguagem Java, abordando a modelagem de classes, herança para reaproveitamento de código e o uso de interfaces e polimorfismo para a criação de sistemas flexíveis

> Este repositório foca na aplicação da OOP através do projeto ScreenMatch e de desafios, assim consolidando conceitos de encapsulamento, modificadores de acesso e a estruturação de uma aplicação de áudio com lógica de negócios aplicada.
